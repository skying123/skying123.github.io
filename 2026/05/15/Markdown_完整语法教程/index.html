Markdown 完整语法教程

Markdown 是一种轻量级标记语言，由 John Gruber 于 2004 年创建。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的 HTML 文档。

编写建议

  1. 保持一致性：选择一种列表标记风格（- 或 *）并始终使用
  2. 空行分隔：段落之间始终保留空行，提高可读性
  3. 标题层级：不要跳级使用标题（如 H1 后直接 H3）
  4. 代码块标注语言：始终为代码块指定语言以获得语法高亮
  5. 链接管理：长文档建议使用引用式链接，将链接定义放在文档末尾
  6. 图片替代文本：始终为图片提供有意义的 alt 文本，提高可访问性
  7. 表格对齐：使用对齐标记让表格更易读

提示：Markdown 的核心设计哲学是「易读易写」。在编写时，优先保证纯文本的可读性，标记符号应该尽可能不干扰内容的阅读。
