七、常见问题排查

7.1 hexo: command not found

7.2 node_modules 损坏

7.3 Deployer not found: git

7.4 图片 404

  1. 确认图片放在 source/img/，不是 public/img/ 或 themes/
  2. 确认 _config.yml 中 url 配置正确
  3. 确认文件名和扩展名完全一致（区分 favicon.ico 和 favicon.png）
  4. 执行 hexo clean && hexo d

7.5 Git 权限错误（.deploy_git 所有者不一致）

7.6 页面一直加载中（标签/分类/友链/关于）

7.7 深色模式卡片还是白色

  • 确认 darkmode.enable: true
  • 点击页面上的 🌞/🌜 切换按钮切换到 Dark 模式
  • 或通过自定义 CSS 强制深色卡片样式

八、常用命令速查

命令 作用
hexo new "标题" 新建文章
hexo new page "名称" 新建独立页面
hexo s 启动本地预览
hexo g 生成静态文件
hexo d 部署到远程（含生成）
hexo clean 清理缓存
hexo clean && hexo d 完整部署流程
