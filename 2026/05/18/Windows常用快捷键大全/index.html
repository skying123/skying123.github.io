Windows 常用快捷键大全

系统整理 Windows 10/11 最实用的键盘快捷键，涵盖系统操作、文件管理、窗口控制、文本编辑、截图录屏、命令行及高级功能，助你大幅提升操作效率。

一、通用系统快捷键

1.1 最常用核心快捷键

快捷键 功能说明 使用场景
Ctrl + C 复制选中内容 文本、文件、图片等
Ctrl + X 剪切选中内容 移动文件或文本
Ctrl + V 粘贴内容 配合复制/剪切使用
Ctrl + Z 撤销上一步操作 误删、误改时救命
Ctrl + Y 重做（恢复撤销） 撤销过头了回退
Ctrl + A 全选 选中当前区域所有内容
Ctrl + S 保存 随时保存，防止丢失
Ctrl + F 查找 在文档/网页中搜索
Ctrl + P 打印 快速调出打印对话框
Ctrl + N 新建 新建窗口/文档/文件夹
Ctrl + O 打开 打开文件对话框
Ctrl + W 关闭当前窗口/标签页 比鼠标点 × 更快
Ctrl + Shift + N 新建文件夹 在资源管理器中
Ctrl + Shift + T 恢复刚关闭的标签页 浏览器中极其实用
Ctrl + Shift + Esc 直接打开任务管理器 系统卡死时首选
Ctrl + Alt + Delete 打开安全选项界面 可锁屏/切换用户/任务管理器
Alt + F4 关闭当前程序/窗口 快速退出应用
Alt + Tab 切换最近使用的窗口 按住 Alt 反复按 Tab 选择
Alt + Shift + Tab 反向切换窗口 往回切换
Alt + Enter 查看文件属性 选中文件后按
Alt + 空格键 打开当前窗口系统菜单 可还原/移动/关闭窗口
F2 重命名选中项 文件/文件夹快速改名
F3 搜索/查找下一个 资源管理器中搜索
F5 刷新 刷新网页/文件夹内容
F11 全屏模式 浏览器/资源管理器
Esc 取消/关闭弹窗/退出全屏 万能取消键

二、Win 键组合大全

Win 键即键盘上带有 Windows 徽标的按键，是 Windows 最高效的快捷入口。

2.1 系统操作类

快捷键 功能说明
Win 打开/关闭开始菜单
Win + A 打开操作中心（通知面板）
Win + I 打开 Windows 设置
Win + L 立即锁定电脑
Win + D 显示桌面（再按一次恢复）
Win + M 最小化所有窗口
Win + Shift + M 还原最小化的窗口
Win + E 打开文件资源管理器
Win + R 打开运行对话框
Win + U 打开辅助功能/轻松使用设置
Win + Pause 打开系统属性（Win11 需额外设置）
Win + S / Win + Q 打开搜索/搜索应用和文件
Win + X 打开快速链接菜单（相当于右键开始按钮）
Win + V 打开剪贴板历史记录（需提前开启）
Win + . / Win + ; 打开表情符号面板 😄
Win + K 打开快速连接（投屏/蓝牙设备）
Win + G 打开 Xbox Game Bar（录屏/性能监控）
Win + H 打开语音输入（Win11 增强）

2.2 窗口管理类

快捷键 功能说明
Win + ← 将窗口贴靠到屏幕左侧
Win + → 将窗口贴靠到屏幕右侧
Win + ↑ 最大化窗口
Win + ↓ 最小化窗口 / 还原窗口
Win + Shift + ←/→ 将窗口移动到另一显示器
Win + Home 最小化除当前窗口外的所有窗口
Win + 数字键 (1-9) 打开任务栏上对应位置的应用
Win + Shift + 数字键 打开对应应用的新实例
Win + Ctrl + 数字键 切换到对应应用的最后一个活动窗口
Win + Alt + 数字键 打开对应应用的跳转列表
Win + T 在任务栏应用间循环切换
Win + Tab 打开任务视图（时间线/虚拟桌面）
Win + Ctrl + D 新建虚拟桌面
Win + Ctrl + F4 关闭当前虚拟桌面
Win + Ctrl + ←/→ 在虚拟桌面间切换

2.3 截图与辅助类

快捷键 功能说明
Win + PrtScn 全屏截图并自动保存到图片文件夹
Win + Shift + S 打开截图工具栏（矩形/任意/窗口/全屏）
Win + + 打开放大镜并放大
Win + - 放大镜缩小
Win + Esc 关闭放大镜
Win + Ctrl + C 打开颜色滤镜（需提前开启）
Win + Ctrl + Enter 打开讲述人（屏幕阅读器）

三、文件资源管理器

在文件资源管理器（此电脑/文件夹窗口）中使用的快捷键：

快捷键 功能说明
Ctrl + N 打开新的资源管理器窗口
Ctrl + W 关闭当前窗口
Ctrl + Shift + N 新建文件夹
Ctrl + E / Ctrl + F 定位到搜索框
Ctrl + Shift + E 展开所有文件夹（导航窗格）
Ctrl + 鼠标滚轮 更改文件和文件夹图标大小
Ctrl + Shift + 数字键 以指定大小显示图标（1=超大，8=平铺）
Alt + ↑ 返回上一级文件夹
Alt + ← 后退
Alt + → 前进
Alt + D / F4 选中地址栏
F3 搜索框
F4 显示地址栏下拉列表
F5 刷新当前窗口
F6 在窗口元素间循环切换
F11 全屏模式
Num Lock + * 展开选中文件夹下的所有子文件夹
Num Lock + + 展开选中文件夹
Num Lock + - 折叠选中文件夹
Ctrl + Shift + 1/2/3... 切换视图方式（超大/大/中等/小图标/列表/详细信息/平铺/内容）

四、窗口与任务管理

4.1 窗口快速操作

快捷键 功能说明
Win + D 一键显示桌面
Win + M 最小化所有窗口
Win + Shift + M 还原所有最小化窗口
Win + Home 仅保留当前窗口，其余最小化
Win + 方向键 窗口贴靠分屏（左/右/上最大化/下最小化）
Win + Shift + 上 垂直拉伸窗口
Win + Shift + 下 还原/最小化垂直拉伸的窗口
Alt + Tab 切换窗口（按住 Alt，按 Tab 选择）
Alt + Esc 按打开顺序切换窗口
Alt + F4 关闭活动窗口
Alt + 空格 + N 最小化窗口
Alt + 空格 + X 最大化窗口
Alt + 空格 + R 还原窗口
Alt + 空格 + M 移动窗口（方向键控制）
Alt + 空格 + S 调整窗口大小（方向键控制）

4.2 任务管理器操作

打开任务管理器后：

快捷键 功能说明
Ctrl + Shift + Esc 直接打开（绕过安全界面）
Ctrl + Alt + Delete → 选择 间接打开
Tab / Shift + Tab 在任务管理器各区域间切换
Alt + E 结束选中进程
Del 结束选中进程
Ctrl + Shift + N 运行新任务

五、文本编辑通用快捷键

适用于记事本、Word、浏览器输入框、聊天软件等绝大多数文本输入场景：

快捷键 功能说明
Ctrl + A 全选
Ctrl + C 复制
Ctrl + X 剪切
Ctrl + V 粘贴
Ctrl + Shift + V 纯文本粘贴（无格式）
Ctrl + Z 撤销
Ctrl + Y 重做
Ctrl + F 查找
Ctrl + H 替换（部分应用）
Ctrl + G 转到指定行/位置
Ctrl + B 粗体
Ctrl + I 斜体
Ctrl + U 下划线
Ctrl + Shift + > 增大字号
Ctrl + Shift + < 减小字号
Ctrl + 方向键 按词/段跳跃移动光标
Ctrl + Home 跳到文档开头
Ctrl + End 跳到文档末尾
Shift + 方向键 逐字/逐行选中
Ctrl + Shift + 方向键 按词/段选中
Shift + Home 选中到行首
Shift + End 选中到行尾
Ctrl + Backspace 删除前一个单词
Ctrl + Delete 删除后一个单词
Ctrl + Shift + L 选中所有匹配项（VS Code 等编辑器）
Ctrl + D 选中下一个匹配项（VS Code）
Ctrl + / 注释/取消注释（代码编辑器）
Tab 缩进 / 焦点跳转
Shift + Tab 反向缩进 / 反向跳转
Ctrl + Enter 在下方插入新行（Word/编辑器）
Ctrl + Shift + Enter 在上方插入新行
Ctrl + Shift + ↑/↓ 上下移动当前行（部分编辑器）
Ctrl + J 文本两端对齐（Word）
Ctrl + L 左对齐
Ctrl + R 右对齐
Ctrl + E 居中对齐

六、浏览器常用快捷键

适用于 Chrome、Edge、Firefox 等主流浏览器：

快捷键 功能说明
Ctrl + T 新建标签页
Ctrl + W / Ctrl + F4 关闭当前标签页
Ctrl + Shift + T 恢复刚关闭的标签页
Ctrl + Tab 切换到下一个标签页
Ctrl + Shift + Tab 切换到上一个标签页
Ctrl + 数字键 (1-8) 切换到对应位置标签页
Ctrl + 9 切换到最后一个标签页
Ctrl + N 新建浏览器窗口
Ctrl + Shift + N 新建无痕/隐私窗口
Ctrl + D 将当前页加入收藏夹
Ctrl + Shift + D 将所有打开标签页加入收藏夹
Ctrl + H 打开历史记录
Ctrl + J 打开下载记录
Ctrl + F 在页面中查找
Ctrl + G / F3 查找下一个
Ctrl + Shift + G / Shift + F3 查找上一个
Ctrl + R / F5 刷新页面
Ctrl + Shift + R / Ctrl + F5 强制刷新（忽略缓存）
Ctrl + + 放大页面
Ctrl + - 缩小页面
Ctrl + 0 恢复默认缩放比例
Ctrl + L / Alt + D / F6 聚焦到地址栏
Ctrl + K 聚焦到搜索框（部分浏览器）
Ctrl + Enter 自动补全 www. 和 .com
Alt + Home 打开主页
Alt + ← 后退
Alt + → 前进
Space / Shift + Space 向下/向上滚动一页
Home 跳到页面顶部
End 跳到页面底部
Ctrl + P 打印页面
Ctrl + S 保存网页
Ctrl + Shift + Delete 清除浏览数据
F11 全屏模式
F12 / Ctrl + Shift + I 打开开发者工具
Ctrl + Shift + J 打开开发者工具控制台
Ctrl + U 查看网页源代码

七、截图与辅助功能

7.1 系统自带截图

快捷键 功能说明
PrtScn (Print Screen) 截取全屏到剪贴板
Win + PrtScn 截取全屏并自动保存到 图片/屏幕截图 文件夹
Alt + PrtScn 截取当前活动窗口到剪贴板
Win + Shift + S 打开截图工具栏（矩形/任意形状/窗口/全屏）
Win + G 打开 Xbox Game Bar（可录屏、截屏、查看性能）
Win + Alt + PrtScn 录制/截取当前游戏/应用窗口（Game Bar）
Win + Alt + R 开始/停止录屏（Game Bar）

7.2 放大镜与辅助功能

快捷键 功能说明
Win + + 打开放大镜并放大
Win + - 放大镜缩小
Win + Esc 关闭放大镜
Ctrl + Alt + I 反色显示（高对比度）
Ctrl + Alt + D 切换到停靠视图（放大镜）
Ctrl + Alt + F 切换到全屏视图（放大镜）
Ctrl + Alt + L 切换到镜头视图（放大镜）
Win + Ctrl + Enter 打开讲述人（屏幕阅读器）
Win + Ctrl + C 打开颜色滤镜
Win + U 打开轻松使用设置中心
Shift（连按 5 次） 打开粘滞键提示
Num Lock（连按 5 次） 打开切换键提示

适用于 CMD、PowerShell、Windows Terminal：

快捷键 功能说明
Ctrl + C 终止当前命令
Ctrl + V / Ctrl + Shift + V 粘贴（部分终端需后者）
Ctrl + A 全选（Windows Terminal）
Ctrl + Shift + C 复制（Windows Terminal）
Ctrl + L 清屏
Ctrl + ↑/↓ 逐行滚动查看输出
Tab 自动补全命令/路径
Tab（连按） 在补全候选间循环
Shift + Tab 反向循环补全候选
↑ / ↓ 浏览历史命令
F3 自动输入上一个以某字符开头的命令
F7 显示命令历史列表（CMD）
F8 搜索历史命令（CMD）
F9 按编号运行历史命令（CMD）
Esc 清空当前行
Ctrl + ←/→ 按单词移动光标
Ctrl + Home 光标移到行首
Ctrl + End 光标移到行尾
Ctrl + Shift + ↑/↓ 调整窗口透明度（部分终端）
Alt + Enter 全屏模式（CMD）
Ctrl + Shift + + 放大字体（Windows Terminal）
Ctrl + - 缩小字体（Windows Terminal）
Ctrl + 0 恢复默认字体大小
Ctrl + Shift + 1/2/3... 打开对应配置文件的终端标签
Ctrl + Shift + W 关闭当前标签
Ctrl + Tab 切换终端标签
Ctrl + Shift + P 打开命令面板（Windows Terminal）

九、高级与开发者快捷键

9.1 文件与程序高级操作

快捷键 功能说明
Shift + 右键点击文件 显示扩展右键菜单（如”复制为路径”）
Shift + 右键点击文件夹 显示”在此处打开命令窗口/PowerShell”
Shift + 删除 永久删除（不经过回收站）
Ctrl + Shift + 拖动文件 创建快捷方式
Alt + 双击文件 查看属性
Ctrl + Shift + 双击程序 以管理员身份运行
Win + R → 输入命令 快速启动程序/功能

Win + R 常用命令：

命令 打开的功能
notepad 记事本
calc 计算器
mspaint 画图
control 控制面板
appwiz.cpl 程序和功能（卸载程序）
taskmgr 任务管理器
regedit 注册表编辑器
cmd 命令提示符
powershell PowerShell
wt Windows Terminal
explorer 文件资源管理器
ms-settings: Windows 设置
devmgmt.msc 设备管理器
diskmgmt.msc 磁盘管理
services.msc 服务管理
eventvwr 事件查看器
cleanmgr 磁盘清理
perfmon 性能监视器
resmon 资源监视器
osk 屏幕键盘
mstsc 远程桌面连接
snippingtool 截图工具
msinfo32 系统信息
dxdiag DirectX 诊断工具

9.2 多显示器操作

快捷键 功能说明
Win + Shift + ←/→ 将当前窗口移动到左/右显示器
Win + P 切换投影模式（仅电脑/复制/扩展/仅第二屏幕）

9.3 BIOS/UEFI 与启动

快捷键 功能说明
Shift + 点击重启 进入高级启动选项（可进安全模式/UEFI）
Win + Ctrl + Shift + B 重置显卡驱动（解决黑屏/显示问题）

十、快捷键记忆技巧

10.1 规律总结

修饰键 一般含义 记忆法
Ctrl 控制基础操作 Control
Alt 交替/辅助功能 Alternate
Shift 反向/增强 如反向 Tab、永久删除
Win 系统级功能 Windows 专属
Ctrl + Shift 强力版基础操作 比 Ctrl 更强
Win + Ctrl 虚拟桌面/系统核心 深层系统功能

10.2 字母联想记忆

键位 功能联想
C = Copy 复制
V = 形似粘贴符号 粘贴
X = 剪刀形状 剪切
Z = Zigzag（回退） 撤销
S = Save 保存
F = Find 查找
P = Print 打印
N = New 新建
O = Open 打开
W = Window 关闭窗口
E = Explorer 资源管理器
R = Run 运行
L = Lock 锁屏
D = Desktop 桌面
T = Tab 标签页
M = Minimize 最小化
I = Settings 设置（形似齿轮？）
A = Action/All 操作中心/全选
K = Konnect 连接设备
G = Game 游戏栏
H = Hear 语音输入
U = Utility 辅助功能

10.3 方向键组合规律

  • Win + 方向键 = 窗口贴靠/最大化/最小化
  • Ctrl + 方向键 = 光标按词/段移动
  • Shift + 方向键 = 选中文本
  • Ctrl + Shift + 方向键 = 按词/段选中
